# webpack-demo-app

## "What is Webpack?"

This repository is an exercise in installing Webpack, configuration and adding loaders for CSS, SASS, HTML, JS and asset files.

This will also exercise cache busting, setting up a dev server and splitting development and production config files.

Incorporating multiple webpack plugins including:
 - clean-webpack-plugin
 - mini-css-extract-plugin
 - optimize-css-assets-webpack-plugin
 - terser-webpack-plugin
 - html-webpcak-plugin
